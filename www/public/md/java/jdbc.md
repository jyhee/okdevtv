# JDBC
* Java Database Connectivity

## Basic Process
1. get JAR file for each DB
2. Prepare username, password, jdbcurl for db
3. Create `Connection`
4. Create `PreparedStatement`
5. Get `Result` from 4.
6. Work with `Result`
7. Close `Result`
8. Close `PreparedStatement`
9. Close `Connection`
