# pyenv
* Simple Python version management
* https://github.com/pyenv/pyenv

## install
* `xcode-select --install`
* `brew install pyenv`

## version install
* `pyenv install 3.5.2`

## commands
```
pyenv commands
pyenv local
pyenv global
pyenv shell
pyenv install
pyenv uninstall
pyenv rehash
pyenv version
pyenv versions
pyenv which
pyenv whence
```
